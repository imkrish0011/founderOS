
# FounderOS Curriculum

> Phase I — Foundations

## Goal
Build a rock-solid foundation in programming, mathematics, tooling, and problem solving before touching Machine Learning.

**Estimated Duration:** 8–12 Weeks  
**Difficulty:** ⭐☆☆☆☆

# Module 1 — Development Environment

## Topic 1 — Visual Studio Code

### Why
A professional AI engineer spends most of their day inside development tools. Learning them early removes unnecessary friction later.

### Learn
- Interface
- Explorer
- Extensions
- Integrated Terminal
- Debugger
- Search
- Workspace
- Settings
- Profiles

### Resources

**Documentation**
- https://code.visualstudio.com/docs

**Video**
- freeCodeCamp — VS Code Full Course

### Practice
- Install VS Code
- Configure Python extension
- Configure Git integration
- Learn the debugger

**Checkpoint**
- Build and debug a Python project inside VS Code.
