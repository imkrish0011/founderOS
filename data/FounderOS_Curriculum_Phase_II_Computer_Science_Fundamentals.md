
# FounderOS Curriculum

# Phase II — Computer Science Fundamentals

> Goal: Develop the problem-solving and computer science foundations required for AI engineering and large-scale software systems.

**Estimated Duration:** 10–14 Weeks  
**Difficulty:** ⭐⭐☆☆☆

# Module 1 — Computational Thinking

## Learn
- Problem decomposition
- Pattern recognition
- Abstraction
- Algorithms
- Flowcharts
- Pseudocode

## Resources
### Video
- Harvard CS50 (Weeks 0–2)

### Practice
- Write pseudocode for 20 everyday problems.
- Convert pseudocode into Python.

## Learn

- Big Omega
- Big Theta

- Amortized analysis (intro)

## Resources
