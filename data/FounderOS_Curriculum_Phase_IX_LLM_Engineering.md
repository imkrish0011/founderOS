
# FounderOS Curriculum

# Phase IX — LLM Engineering

> Goal: Learn how modern Large Language Models are built, adapted, evaluated, and deployed into production AI systems.

**Estimated Duration:** 14–18 Weeks  
**Difficulty:** ⭐⭐⭐⭐⭐

---

# Prerequisites

- Python
- Deep Learning
- NLP
- Transformers
- PyTorch

---

# Module 1 — LLM Fundamentals

## Learn
- What is an LLM?
- Pretraining vs Fine-tuning
- Decoder-only vs Encoder-Decoder
- Scaling Laws
- Context Window
- Tokens & Tokenization

### Resources
- Hugging Face LLM Course
- OpenAI Cookbook
- Andrej Karpathy: Let's Build GPT

### Project
Build a tiny GPT-style language model from scratch.

# Module 2 — Training Pipeline

## Learn
- Dataset Collection
- Data Cleaning
- Tokenizers
- Training Objectives
- Checkpointing
- Mixed Precision
- Distributed Training (Overview)

### Practice
Train a small language model on a custom corpus.
