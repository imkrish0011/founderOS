
# FounderOS Curriculum

# Phase XII — AI Research

> Goal: Transition from implementing existing AI methods to conducting original, reproducible AI research.

**Estimated Duration:** 16–20 Weeks  
**Difficulty:** ⭐⭐⭐⭐⭐

---

# Prerequisites

- Machine Learning
- Deep Learning
- Transformers
- LLM Engineering
- Reinforcement Learning
- Mathematics for AI

---

# Module 1 — Research Foundations

## Learn
- What is scientific research?
- Research lifecycle
- Hypothesis formulation
- Research questions
- Literature review
- Research ethics
- Reproducibility

### Resources
- Stanford CS329S (Machine Learning Systems Design)
- "The Craft of Research"
- Papers With Code

### Project
Write a literature review on one AI topic.

---

# Module 2 — Reading Research Papers

## Learn
- Paper structure
- Abstract analysis
- Method section
- Experiments
- Ablation studies
- Limitations
- Future work

### Practice
Read one paper every week and summarize:
- Problem
- Method
- Results
- Weaknesses
- Ideas for improvement

---

# Module 3 — Experiment Design

## Learn
- Dataset selection
- Baselines
- Metrics
- Hyperparameter search
- Statistical significance
- Error analysis
- Reproducibility checklists

### Tools
- Weights & Biases
- MLflow
- TensorBoard

### Project
Design and document a reproducible benchmark.

---

# Module 4 — Scientific Writing

## Learn
- LaTeX
- Citations (BibTeX)
- Figures & Tables
- Writing methodology
- Writing results
- Writing discussions

### Tools
- Overleaf
- Zotero
- Mendeley

### Project
Write a conference-style paper based on one of your AI projects.

---

# Module 5 — Reproducing Landmark Papers

## Reproduce

- AlexNet
- ResNet
- BERT
- Vision Transformer
- GPT (small scale)
- DQN

Document:
- Differences
- Challenges
- Improvements

---

# Module 6 — Open Source Research

## Learn
- Reading GitHub repositories
- Reproducing code
- Benchmarking
- Contributing to open source
- Writing technical documentation

### Project
Submit at least one accepted pull request to an AI project.

---

# Module 7 — Research Portfolio

Build a portfolio containing:

- Paper summaries
- Reproductions
- Novel experiments
- Benchmark reports
- Conference-style reports
- Open-source contributions

# Major Projects

## Beginner
- Weekly Paper Summaries

## Intermediate
- Reproduce a published paper

## Advanced
Conduct original research:
- Identify a gap
- Propose a method
- Run experiments
- Write a paper
- Publish code

---

# Weekly Goals

- Read 2 papers
- Reproduce 1 experiment
- Maintain research notes
- Push research code to GitHub

## Skills
- [ ] Literature Review
- [ ] Experiment Design
- [ ] Statistical Analysis
- [ ] Scientific Writing
- [ ] Paper Reproduction

## Projects
- [ ] Paper Summaries

## Completion Requirement

You should be capable of independently reading current AI literature, reproducing published work, designing rigorous experiments, and writing research papers suitable for submission or public release.
