
# FounderOS Curriculum

# Phase VI — Computer Vision

> Goal: Master computer vision from image fundamentals to modern vision models capable of solving real-world problems.

**Estimated Duration:** 12–16 Weeks  
**Difficulty:** ⭐⭐⭐⭐☆

# Prerequisites

- Python
- Machine Learning
- Deep Learning
- PyTorch
- Linear Algebra
