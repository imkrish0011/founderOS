
# FounderOS Curriculum

# Phase VIII — Transformers

> Goal: Master the Transformer architecture that powers modern language models, vision models, and multimodal AI systems.

**Estimated Duration:** 12–16 Weeks  
**Difficulty:** ⭐⭐⭐⭐⭐

---

# Prerequisites

- Python
- Deep Learning
- NLP
- PyTorch
- Linear Algebra

---

# Module 1 — Why Transformers?

## Learn
- Limitations of RNNs and LSTMs
- Parallel Processing
- Long-Range Dependencies
- Sequence Modeling

### Resources
- Illustrated Transformer
- Stanford CS25 (Transformers)
- Hugging Face Course

# Module 2 — Attention Mechanism

## Learn
- Query, Key, Value
- Scaled Dot-Product Attention
- Self-Attention
- Cross-Attention
- Multi-Head Attention

### Practice
Implement scaled dot-product attention in PyTorch.

# Module 3 — Transformer Architecture

## Learn
- Positional Encoding
- Encoder
- Decoder
- Feed Forward Network
- Residual Connections
- Layer Normalization

### Project
Build a miniature Transformer encoder from scratch.

# Module 4 — Training Transformers

## Learn
- Tokenizers
- Vocabulary
- Masking

- Training Loops
- Mixed Precision

Train a tiny language model on a custom text dataset.
