
# FounderOS Curriculum

# Phase III — Mathematics for Artificial Intelligence

> Goal: Build the mathematical intuition required to truly understand machine learning, deep learning, optimization, and research papers.

**Estimated Duration:** 12–16 Weeks  
**Difficulty:** ⭐⭐⭐☆☆

---

# Why Mathematics Matters

Mathematics is the language of AI. You are not learning math to solve textbook problems—you are learning it to understand algorithms, optimize neural networks, read research papers, and build new models.

---

# Module 1 — Linear Algebra

## Learn
- Scalars, Vectors, Matrices, Tensors
- Matrix Operations
- Matrix Multiplication
- Identity & Inverse
- Rank
- Determinant
- Eigenvalues & Eigenvectors
- Singular Value Decomposition (Intro)

### Resources
- 3Blue1Brown — Essence of Linear Algebra
- MIT OpenCourseWare (Linear Algebra)
- Khan Academy

### Practice
- Solve 50 matrix problems
- Implement matrix multiplication in Python
- Implement Gaussian Elimination

### Project
Create a Matrix Calculator using NumPy.

Checkpoint:
Explain why embeddings are vectors.
